## FAQ  (Frequently Asked Questions)

### Q: How do I install and enable the extension?  
**A:** Follow the instructions in the **Installation** section of the README. You’ll load the extension in developer mode and then toggle styles using the popup interface.

### Q: Can I enable multiple styles at the same time?  
**A:** Yes! The extension allows you to toggle multiple styles simultaneously through the popup interface. However, please note that some styles may conflict with each other because they modify the same parts of YouTube’s layout (for example, "Centered Video Player" and "Comments on the Right"). In such cases, enabling conflicting styles may cause layout issues or unexpected behavior.

### Q: Will this extension work on mobile devices or tablets?  
**A:** No. The extension is designed and optimized specifically for the desktop version of YouTube. Mobile layouts are different and may not work.

### Q: What should I do if YouTube updates break the extension or the layout looks wrong?  
**A:** YouTube frequently updates its site, which can sometimes cause layout issues. If something breaks, disable the affected style and report the issue on GitHub. I actively update the extension to keep it compatible.

### Q: Will these YouTube customizations slow things down?
**A**: **Not really.**  The changes are mostly visual—like styles, animations, and button tweaks.  
They don’t affect how YouTube works, and performance should stay smooth on most devices.

### Q: Can I disable all styles to return to the default YouTube look?  
**A:** Absolutely. Simply open the popup and turn off all the toggles. If the layout doesn’t reset, reload the page manually.

### Q: Does this extension support themes or dark mode customization?  
**A:** **Not directly**, but it's fully compatible with YouTube’s dark / light mode. Currently, the extension focuses on layout and UI improvements. Dark mode and themes are managed by YouTube or other extensions.

### Q: Does this extension collect or send any personal data?  
**A:** No. This extension only modifies the YouTube user interface locally in your browser. It does not collect, store, or transmit any personal information.

### Q: Does the extension work only on Chrome?  
**A:** This extension works on Chrome and every Chromium fork. It doesn't work on Mozilla Firefox.

### Q: Can I contribute to the project or suggest new features?  
**A:** Yes! Contributions, bug reports, and feature requests are welcome. Please open an issue or submit a pull request on GitHub.
